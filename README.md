# web-am-radio-sound-simulator

A web-based audio effect simulator that recreates the classic sound of AM radio.

**[Try it here](https://e04.github.io/web-am-radio-sound-simulator/dist/index.html)**

## Features

- Real-time AM radio effect processing
- Audio visualization with oscilloscope
- Adjustable parameters for customizing the sound

## Usage Notes

On macOS, if you want to route system audio (e.g., music from other apps) through this simulator, you'll need to install [BlackHole](https://github.com/ExistentialAudio/BlackHole) to create a virtual audio device.

On Windows, you can use [VB-Audio Virtual Cable](https://vb-audio.com/Cable/) or [Voicemeeter](https://vb-audio.com/Voicemeeter/) for similar functionality.

## Development

npm install
npm run dev
